# AOS — Core Philosophy

AOS follows the operating chain:

```text
chat -> decision -> file -> folder -> tracker -> memory -> reusable system
```

The goal is not simple automation. The goal is to build a new operating model for business where AI and humans work through durable structure, reusable systems, and file-first truth.

The brain stores durable retrieval memory. The folder system remains the human-readable source of truth.
